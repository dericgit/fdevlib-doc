ui-datepicker, ui-datepicker-time : 日历控件

介绍

该组件提供日历和时间选择功能。

版本

  • Denis: 从KISSY移植 ---- 2011.03.01

开始

//日期选择功能                
$.use('ui-datepicker', function(){
    $('...').datepicker(options);
});
//日期+时间选择功能                
$.use('ui-datepicker-time', function(){
    $('...').datepicker({
        showTime: true,
        //...
    });
});

Configs

date : Date
Default : new Date()
默认高亮（红色加粗字体）的当前日期
startDay : Number
Default : 0
日历中一周的第一天，"0"表示周日。
pages : Number
Default : 1
同时显示几个月的日历
popup : Bolean
Default : true
是否以浮出层的方式显示日历
closable : Bolean
Default : false
在"popup"配置为false的时候，允许日历在选择完日期后关闭。
rangeSelect : Bolean
Default : false
是否提供区间选择的功能
minDate : Date | false
Default : false
提供日历可选择范围的起始日期
maxDate : Date | false
Default : false
提供日历可选择范围的截止日期
range : Object
Default : { start: null, end: null }
提供日历可选择范围的截止日期
navigator : Bolean
Default : true
提供标题栏修改年月的功能
shim : Bolean
Default : false
是否在IE6下提供垫片
showTime : Bolean
Default : false
是否提供时分秒选择的扩展功能需要"ui-datepicker-time"
triggerType : String
Default : 'click'
触发日历的事件在input上建议使用'blur'

Events

beforeShow : undefined
  • return : Bolean 返回"flase"中止日历激活
日历打开前触发本事件，返回"false"来中止。
rangeSelect : undefined
  • e : Event 事件对象
  • ui : Object 提供选择的日期范围（起始/结束），精确到日。
    • start : Date 起始日期
    • end : Date 截止日期
选中日期范围后触发。
select : undefined
  • e : Event 事件对象
  • ui : Object 提供选择的日期，精确到日。
    • date : Date 选中的日期
当"rangeSelect"为"undefined"，选中日期后触发。
monthChange : undefined
  • e : Event 事件对象
  • ui : Object 提供选择的月份的第一天，精确到月。
    • date : Date 选中的月份的第一天
改变月份后触发。
timeSelect : undefined
  • e : Event 事件对象
  • ui : Object 提供选择的日期，精确到秒。
    • date : Date 选中的时间
当使用"ui-datepicker-time"扩展时，选中时间后触发。
