ui-dialog : 浮出层

介绍

  • 提供模拟对话框、浮出层遮罩功能
  • 多实例化，互不干扰；共用遮罩，屏蔽遮罩层下方Tab事件，遮罩层上方Tab循环
  • 提供shim配置项，兼容IE6，完美遮挡select元素

版本

  • 汤汤: 修复IE6短文档页面遮罩无法覆盖全页面的BUG ---- 2011.03.22

开始

$.use('ui-dialog', function(){
    $('...').dialog(options);
});

Statics

jQuery.ui.dialog.zIndex : Number
Default : 2000
浮出层z-index样式的自增基数

Configs

fixed : Boolean
Default : false
浮出层是否跟随滚动
center : Boolean
Default : false
浮出层是否打开居中（水平+垂直）
draggable : Boolean | Object
Default : false
是否支持浮出层可拖动。设为 true 只提供默认的拖动效果，可以设置Draggable组件支持的配置Object来自定义拖动效果。
draggable: {
    handle: 'div.header',
    containment: 'body',
    start: startDragHandler
}
modal : Boolean
Default : true
浮出层是否带有蒙版效果
shim : Boolean
Default : false
是否在IE6下使用iframe垫片。注意：该配置项依赖于ui-core模块，否则无法生效！
modalCSS : Object
自定义蒙版层的样式
css : Object
自定义浮出层容器的样式
fadeIn : Number
Default : false
浮出层打开时的渐入效果时间(ms)
fadeOut : Number
Default : false
浮出层关闭时的渐出效果时间(ms)
timeout : Number
Default : 0
延时自动关闭浮出层的时间(ms)

Events

open
浮出层打开后触发
beforeClose
浮出层关闭前触发
close
浮出层关闭后触发
