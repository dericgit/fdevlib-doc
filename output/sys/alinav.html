Alinav 1688 页尾站内导航

介绍

在页面 footer 的上方，通常会有一个站内的导航区块，这个区块叫做 alinav。该区块的设计样式全站风格统一，分灰色、黑色背景两种，不允许重新设计。

alinav 的设计风格是统一的，但是内部的链接内容、右侧的图片URL 是允许业务方自定义的，故从组件的层面只提供统一的 CSS 文件引用，及可供拷贝的 Demo HTML。

CSS 文件不允许本地拷贝，或者覆盖样式，只允许使用统一的这份引用。

使用说明

  • alinav目前由前端通用负责维护
  • 目前全站尾部导航规范请移步 全站尾部导航规范
  • alinav对应的CSS文件需要产品自己引入：
     <link href="http://style.c.aliimg.com/sys/css/universal/alinav/standard.css" rel="stylesheet"/>

注意：

  • alinav 组件 CSS 采用的是自适应间距布局的实现，所以站内导航可以定义3列、4列或是5列，样式上能够做到自动均匀分配，使用者只需关注具体链接内容即可。
  • 灰色背景与黑色背景的 HTML 结构是完全一致的，唯一的不同是黑色背景Demo中需要在 alinav 的外层 DIV nav-container 上再添加一个 class “nav-black”
