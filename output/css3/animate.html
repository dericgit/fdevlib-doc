Animate.css

介绍

Animate.css包含了一堆很酷、有趣、跨浏览器的动画，你可以方便在项目中使用。

版本

  • 目前引的线上的animate.css库
  • 后续会将animate.css按动画类型拆分，拆分成各个小文件。并结合其他库或者一些动画效果产出新的动画类型。

开始

引入CSS

// 文件暂未上传
@import url("animate.min.css");

addClass

$('#element').addClass('animated bounceOutLeft');

duration、delay、times

/* vendor为浏览器前缀 */
#element {
    -vendor-animation-duration: 3s;
    -vendor-animation-delay: 2s;
    -vendor-animation-iteration-count: infinite;
}
